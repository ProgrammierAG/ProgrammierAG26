#### Aufgabenstellung

Erstelle ein Programm, das den Nutzer zur Eingabe eines Passworts auffordert. Anschließend soll überprüft werden, ob das Passwort mindestens acht Zeichen lang ist. Ist das Passwort kürzer, wird eine ``Exception`` mit der Meldung „Zu kurzes Passwort!“ ausgelöst. Hat das Passwort die erforderliche Länge, wird es in der Konsole ausgegeben.

---

#### Konsole

```
Gebe ein Passwort ein: 123
Zu kurzes Passwort

Gebe ein Passwort ein: GutesPasswort
GutesPasswort
```
