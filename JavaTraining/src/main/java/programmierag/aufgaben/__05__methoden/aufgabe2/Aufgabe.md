#### Aufgabenstellung

Erstelle eine Methode mit zwei `int`-Parametern `min` und `max`, die eine zufällige Zahl zurückgibt, wobei die beiden Grenzwerte eingeschlossen sind. 
Danach erstelle ein Programm, das so lange läuft, bis die Maximalzahl erreicht wurde. Anschließend soll `max` um 1 erhöht werden und das Programm erneut versuchen, die Maximalzahl 3 mal zu erraten.
