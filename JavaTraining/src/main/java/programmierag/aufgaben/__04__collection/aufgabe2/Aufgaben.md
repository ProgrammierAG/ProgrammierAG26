#### Aufgabenstellung

Erstelle ein Programm, das mehrere Namen annimmt und diese anschließend sortiert: zuerst alphabetisch in aufsteigender Reihenfolge, dann in absteigender Reihenfolge und zum Schluss in zufälliger Reihenfolge.

---
#### Konsole

```
Gib mehrere Wörter ein: Anna, Kevin, Ben

Anna
Ben
Kevin

Kevin
Ben
Anna

Kevin 
Anna
Ben
```
