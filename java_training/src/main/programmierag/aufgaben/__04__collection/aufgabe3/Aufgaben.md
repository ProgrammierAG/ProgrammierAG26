#### Aufgabenstellung

Initialisiere ein ``Array<String>``, das mehrere Wörter enthält. Mappe das Array auf ein `Set<Char>`, das nur die letzten Buchstaben der Wörter enthält. Verwende dabei Wörter, die mehrfach denselben letzten Buchstaben haben. Gib anschließend das Array und das Set zurück.

---
#### Konsole

```text
Array: [Baum, Daum]
Set: [m]
```
