#### Aufgabenstellung

Ein Palindrom ist ein Wort, das vorwärts wie auch rückwärts gelesen werden kann.  
Erstelle ein Programm, das überprüft, ob das eingegebene Wort ein Palindrom ist.

---
#### Konsole 

```plainText
Gebe ein Wort ein: Anna
Anna ist ein Palindrom

Gebe ein Wort ein: Jürgen
Jürgen ist kein Palindrom
```
