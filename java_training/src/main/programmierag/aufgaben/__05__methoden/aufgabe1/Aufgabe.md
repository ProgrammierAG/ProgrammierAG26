#### Aufgabenstellung

Erstelle jeweils eine Methode,  + und - wobei jede Methode zwei `double`-Werte annimmt und einen `double`-Wert zurückgibt. Anschließend sollen zwei Zahlen durch die arithmetischen Operationen berechnet werden.

---
#### Konsole

```
Gebe eine Zahl ein: 5
Gebe eine weitere Zahl an: 5

5 + 5 = 10
5 - 5 = 0
```
